How to test Apps through BetaFox

Getting and testing FirefoxOS apps from a developer happens in X steps.

  1. Provision your device
  2. Install the App

Step 1 - Provisioning Your Device

This step looks scary, but it only has to be done once.

WarningWhile a phone is provisioned to be a test device, it will not be able to install or run other apps from the Marketplace.

Requirements:

You need the following pieces to provision your device:

  • USB Cable for your Device
  • Python installed
  • NSS installed
  • adb from the Android SDK or elsewhere
  • ???
Make sure you can see your device. One the command line, try adb devices

If everything worked... Try the following:

./configure-2-distribute-2-gecko.sh {{ publicHostname }}

Your device should reboot and provisioning is finished.

Step 2 - Installing the App

A developer should get you an App install link. Load that link on the browser on your device.

Note: If you have already installed the App, you must remove the app, before installing it a second time.
